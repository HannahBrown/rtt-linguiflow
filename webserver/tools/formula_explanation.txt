METHOD 2

Calculating Word Percent Confidence
~~~~English Sentence~~~~"the"~~~E~~"the"~~~E~~~"the"~~~E~~~~
~~~~Spanish Sentence~~~~"la"~~~S~~~"la"~~~~S~~~"fiesta"~~~S~~~
Pair Score = 6

~~~~English Sentence~~~~"the"~~~E~~"the"~~~E~~~"the"~~~E~~~~
~~~~Spanish Sentence~~~~"la"~~~S~~~"tabla"~~~~S~~~"fiesta"~~~S~~~
Pair Score = 20

the->la = SUM(Pair score*count)/matches = (6*2+1*20)/3=11
the->fiesta = SUM(Pair score*count)/matches =(6*1+20*1)/2 = 13
the->tabla = SUM(Pair score*count)/matches=(20*1)/1=20

**The resulting numbers should be multiplied by a constant 10^x so the results range from 0 to 100%

Formula
Data from en-sp-align_words.sh
Accept: Set percentage to 100%
Reject: Set percentage to 0%, Increase remaining percentages by current_percentage+(current_percentage*removed_percentage)
